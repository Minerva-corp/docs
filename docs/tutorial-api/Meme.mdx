You need at least Node.js version 12.3.0 to install Require package.

## Require package

```bash npm2yarn2pnpm
npm install node-fetch@2
```

## Example Fetch

```typescript ts2esm2cjs|{3}|{3}
import fetch from 'node-fetch';

const fetchMeme = async () {
    const response = await fetch("https://api.minerva.my.id/v1/meme").then(res => res.json());

    console.log(response);
}

fetchMeme();
```
